# Right Reach Backend

This is the backend server for the Right Reach project, built with Node.js and Express.

## Getting Started

1.  Navigate to the `backend` directory:
    ```bash
    cd backend
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```

## API Endpoints

-   `GET /`: Welcome message.
-   `POST /api/auth/register`: Placeholder for patient/doctor registration.
-   `POST /api/auth/login`: Placeholder for user login.
-   `GET /api/patients/:uhid`: Placeholder for retrieving patient records.

## Technology Stack
-   **Server**: Express.js
-   **Environment**: dotenv
-   **Development**: nodemon
-   **Data Serialization**: JSON (Placeholder for Database integration)
